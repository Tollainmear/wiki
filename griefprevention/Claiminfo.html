领地信息

/claiminfo 这个指令会给你展示一个领地的信息，同时也可让你改变一些设置

示例输出内容

Admin Settings

当你可以控制一个领地时，一个管理员设置按钮会显示在领地信息内容上方

DenyMessages

如果设置为 OFF ，即使玩家做一些他们没有权限做的事情，他们也不会收到任何提示消息

ClaimExpiration

设置为 OFF ，该领地不会被领地到期所影响

FlagOverrides

设置为 OFF ，该领地将无视任何适合该种领地的属性

PvPOverride

无视 Sponge config 设置中的PVP设置. 有三种状态:

TRUE - 允许领地内PVP FALSE - 禁止领地内PVP UNDEFINED - 使用 Sponge config 里的设置
